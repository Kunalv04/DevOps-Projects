About This Application

Learn about the technology stack and architecture behind this DevOps educational webapp.

☕

Java

Version 17

Built using Java, a robust and platform-independent programming language perfect for enterprise applications.

🍃

Spring Boot

Version 3.2.0

Leverages Spring Boot's auto-configuration and embedded server capabilities for rapid application development.

📦

Maven

Build Tool

Uses Maven for dependency management, build automation, and project lifecycle management.

🌿

Thymeleaf

Templating Engine

Server-side Java template engine for rendering dynamic HTML pages with natural templating.

Maven Dependencies

spring-boot-starter-web
Web application support with embedded Tomcat
spring-boot-starter-thymeleaf
Server-side HTML templating engine
spring-boot-starter-actuator
Production-ready monitoring and management endpoints
spring-boot-devtools
Development tools with automatic restart
spring-boot-starter-test
Testing support with JUnit and Mockito

How to Run

1. Build the Application

mvn clean package

2. Run the Application

mvn spring-boot:run

3. Access the Application

Open http://localhost:8080 in your browser
